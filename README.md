# Example NextJS dapp
NextJS simple template for creating fast and optimazied dapps.
## Features

- Support for different wallets (Metamask, Coinbase, WalletConnect)
- Contract functions
- Diffren pages with effects, animations, styles.

## Installation
```
npm run dev
```
or
```
yarn dev
```

## Export
```
npm run export
```
or
```
yarn export
```
